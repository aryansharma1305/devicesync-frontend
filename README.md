# DeviceSync Frontend

Marketing landing page for DeviceSync, an AI-powered hardware compatibility platform for IT teams.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```
